Project : https://app.patika.dev/courses/veri-yapilari-ve-algoritmalar/merge-sort-proje

[16,21,11,8,12,22] merge sort will be applied

Steps: 

	       [16,21,11,8,12,22]
      [16,21,11]	   [8,12,22]	     
  [16,21]    [11]       [8,12]    [22]
[16]   [21]  [11]     [8]  [12]   [22]
  [16,21]       [8,11]      [12,22]
      [8,11,16,21]          [12,22]				       
           [8,11,12,16,21,22]
